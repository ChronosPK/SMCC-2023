# SMCC-2023
```
Sibiu Military Cyber Challenge - writeups and dev. configurations
```

<br><br>

## What is SMCC?

```
SMCC is the first CTF competition organized by a military student,
for his fellow colleagues from the military academy and schools in Sibiu, Romania. 
```

```
I came up with this CTF in order to inspire my friends to learn cybersecurity,
to test my skills, and to challenge myself with such an interesting project.
```

```
I started working on it at the beginning of November 2022 and was able to finish all challenges around 15th of January 2023.
After that, the organization phase took off and had to handle multiple things I never would have thought of.
I would have never came this far without a handfull of supporting people, sponsors and teachers.
```

<br><br>

## More details about the competition

```
My team and I gathered 92 particpants with a total of 24 teams.
The CTF that we held consisted of 86 challenges spanning various categories, including 
    Forensics, Cryptography, Web, OSINT, Steganography, Misc, Pentesting, Linux, Windows and Networking & Cybersecurity.
```
