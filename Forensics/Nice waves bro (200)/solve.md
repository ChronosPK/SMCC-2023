## What to do?
This is one of the simplest methods used to hide a message inside an audio file. <br>
I am talking about spectrograms. 

<br>

## How to do it?
In linux, a simple, yet usefull tool I use is `audacity`. <br>
To install it, type in your terminal the following: <br>
```bash
sudo apt install audacity
```
To start it, simply call the binary and the provided `wav` file:
```bash
audacity nice_waves.wav
```

<br>

## Get the job done
You should first play the file. No message can be retrieved this way. <br>
